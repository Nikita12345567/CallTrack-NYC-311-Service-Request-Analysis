# Convert 'Created Date' and 'Closed Date' to datetime with flexible formats
date_formats = ['%m/%d/%Y %I:%M:%S %p', '%m-%d-%y %H:%M']
df['Created Date'] = pd.to_datetime(df['Created Date'], format=date_formats[0], errors='coerce') \
                        .fillna(pd.to_datetime(df['Created Date'], format=date_formats[1], errors='coerce'))

df['Closed Date'] = pd.to_datetime(df['Closed Date'], format=date_formats[0], errors='coerce') \
                        .fillna(pd.to_datetime(df['Closed Date'], format=date_formats[1], errors='coerce'))

# Create 'Request_Closing_Time' as time difference
df['Request_Closing_Time'] = df['Closed Date'] - df['Created Date']

# Convert timedelta to total hours (you can use seconds instead if you prefer)
df['Request_Closing_Time'] = df['Request_Closing_Time'].dt.total_seconds() / 3600  # hours

  1. Statistical Testing

The objective of the hypothesis testing code is to evaluate two key assumptions in the dataset. Test 1 aims to determine whether the average response time differs across various complaint types, using a one-way ANOVA test. The null hypothesis (H) assumes that the average response time is the same for all complaint types, while the alternative hypothesis (H) suggests they are different. Test 2 seeks to assess whether the type of complaint and location (borough) are related, using a Chi-square test for independence. The null hypothesis (H) posits that the complaint type and location are independent, while the alternative hypothesis (H) proposes they are related. These tests help in understanding the relationships between different variables in the data.

Test 1: Whether the average response time across complaint types is similar or not Objective: To determine whether the average response time (represented by Request_Closing_Time) is statistically the same across different types of complaints.

Null Hypothesis (H): There is no significant difference in the average response time across different complaint types. Alternate Hypothesis (H): There is a significant difference in the average response time across different complaint types.

Statistical Test Used: One-way ANOVA (Analysis of Variance)

This test is appropriate because we are comparing the means of a numerical variable (Request_Closing_Time) across more than two independent groups (Complaint Type).

import pandas as pd

)

print(f"P-value: {p_value:.4f}")

This implies that the handling of complaints may be influenced by their type, potentially due to differences in urgency, complexity, or available resources for specific issues.

Chi-Square Test Test 2: Whether there is an association between Complaint Type and Borough

Objective: To determine whether the type of complaint is independent of the borough it was reported in in other words, to test whether certain types of complaints are more common in specific boroughs.

Null Hypothesis (H): There is no association between complaint type and borough. (They are independent.) Alternate Hypothesis (H): There is an association between complaint type and borough. (They are not independent.)

Statistical Test Used: Chi-Square Test of Independence

This test is appropriate because both Complaint Type and Borough are categorical variables, and we are testing for a relationship between them.
